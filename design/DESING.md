# SARA BABY TRACKER & SOUND

* The app provides baby tracking features such as sleep, feeding, and diaper changes. It also
  includes background sounds to help babies relax and fall asleep, as well as a collection of baby
  food recipes.

## 🍼 Features Overview

- Sleep tracking
- Feeding (breast, bottle, solids)
- Diaper changes
- Baby food recipes
- Background sleep sounds

## 🎯 Target Audience

- New moms
- Dads and caregivers
- Families with babies under 2 years old

## 🎨 Design Goals

- Calm and clean UI
- Easy to navigate with one hand
- Soft colors: blue (boy), pink (girl), customizable

## 🌟 Slogan

**"With the Sara App, being a mom is easier."**

---

## 🧪 Tech Stack

### 🖼️ Frontend

- **Flutter** – UI toolkit for crafting natively compiled mobile applications

### 🛠️ Backend

- **Firebase**
    - **Firestore** – NoSQL database for storing structured data
    - **Firebase Auth** – User authentication and management
    - **Cloud Functions** – Serverless backend logic
    - **Storage** – For user-generated media or files

### 🔔 Notifications

- **Firebase Cloud Messaging (FCM)** – Push notifications to iOS and Android devices

### 💾 Local Storage

- **Sqlite** – For storing personal and offline data locally

### 🚀 Deployment

- **App Store** & **Google Play** – For publishing the mobile application

### 🔄 Version Control

- **Git** – Source code management and collaboration

### ⚙️ CI/CD

- **GitHub Actions** – Continuous integration and deployment workflows
